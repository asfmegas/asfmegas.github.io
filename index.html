SADA - TCC

O programa SADA - Sistema de Análise de Desempenho Acadêmico - foi desenvolvido em JavaSE 7 como projeto de conclusão de curso em 2015.1. Ele é um sistema simples que tem como finalidade demonstrar conhecimentos em desenvolvimento de sistemas.

Suas funcionalidades limitam-se ao armazenamento de dados e ao manuseio destes dados. Ele utiliza o sistema MySQL como gerenciador de banco de dados.

Ele foi baseado no formato de notas da faculdade a qual eu estudava, por isso ele possui um formato de gerennciamento de notas bem específico, o que limita seu uso a outras instituições. Mas seu código está disponível para alterações e customizações no repositório do github.com/asfmegas.

Ele pode oferecer alguns benefícios.

  • Falicitar a organização dos seus dados
  • Muito simples de usar
  • Bastante intuitivo

Schedule Tasks

O Schedule Tasks (tradução livre: Agendar Tarefas) é um simples programa para agendamento de tarefas no Linux.

Ele foi desenvolvido para facilitar o agendamento de tarefas, com possibilidades de definir diversas formas de repetições.

Ele foi escrito em Python 3.X e possui um carater mais pedagógico.

Seus parâmetros são os seguintes

  • Name: nome do serviço
  • Notice: para exibir uma notificação quando executado (yes|no)
  • Delete: para excluir o serviço após ele ser executado (yes|no)
  • State: determina o estado do serviço; rodando, parado ou em espera (running|stop)
  • Mode: define a forma do agendamento, podendo ser por data ou por repetições (repeat|date)
  • Repeat: determina o número de repetições (-1 para infinito)
  • Time: define um tempo para cada execução (apenas em modo "repeat")
  • Minute: define o(s) minuto(s) para execução (0-60, sendo 0 todos os minutos)
  • Hour: define a(s) hora(s) para execução (0-24, sendo 0 todas as horas)
  • Day: define o dia para execução (0-31, sendo 0 todos os dias)
  • Month: define o mês para execução (0-12), sendo 0 todos os meses)
  • Command: é a entrada do comando que será executado

Proplus

Este programa escrito em AngularJS possui apenas fins educativos e visa realizar o armazanamento de dados. Ele utiliza o sistema Mongodb como gerenciador banco de dados, ele possui um back-end escrito em NodeJS e é muito simples e prático de usar.

Com ele é possível salvar textos para estudos posteriores. Ele não realiza traduções, até porque, seu objetivo é armazenar os dados e ofercê-los de forma fácil e rápido ao usuário.

Com ele é possível salvar palavras, expressões ou frases para consulta posterior. Assim como textos para realização de traduções manualmente.

Vdown

Vdown é um script em Shell Script que facilita baixar vídeos do YouTube. Com ele você pode criar listas de vídeos é baixá-las quando quiser.

Ele possui como dependência o programa Cclive (v0.7.16 versão atual) que pode ser instalado facilmente no Linux.
