Como não poderia ser diferente, embora relutante em aprender uma nova linguagem de programação, Python foi uma das aprendizagens mais prazerosas que tive. Python se mostrou uma das melhores linguagens que já dispensei meu tempo. Mais isso é uma questão pessoal. Na prática, Python é uma linguagem riquíssima em que possui uma integração muito grande com sistemas Linux. É fácil de aprender, gostosa de usar. Criei um simples programa que realiza o agendamento de tarefas. A princípio, ele roda em sistemas Linux baseado em Debian. Schedule Tasks é um simples programa que permite agendar tarefas em sistemas Linux para ser executado em momento definido pelo usuário. Foi escrito em Python 3.X e possui mas um propósito pedagógico. O código fonte está disponível no meu github. Schedule Tasks é simples de usar. E pretendo detalhar passo a passo o caminho que me levaram ao desenvolvimanto deste pequeno programa. A melhor forma de assimilar uma nova linguagem de programação é praticar seus conceitos, imaginar formas de usar suas ferramentas.
